My Elements

These are my elements

  1. label: allows you to use a caption to an item within the user interface
  2. input: allows user to input information, such as there first name, last name and so on.
  3. mark: allows you to highlight text, can be use to highlight key points, references, and so on.

Used label, input, and mark. The yellow is what mark does! The caption "Name" is what label does, and input is the box, which allows you to type in your name.
